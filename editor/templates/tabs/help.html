Tabs

Adding a new tab

To add a new tab, simply click on the + icon next to the last tab.

Closing tabs

To close a tab simply click on ×

Saving tabs

To save a tab, simply click on CTRL+S or File -> Save. There is also File -> Save All Tabs which ensures that all tabs are saved and that deleted tabs are still deleted upon refresh.

Loading a file

Loading a new file, adds a new tab instead of replacing the contents of the current tab.

Download a file

Downloading a file downloads the contents of the last saved tab.
